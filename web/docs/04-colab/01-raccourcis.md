# ⌨️ Raccourcis pratiques

Voici une sélection des racourcis claviers les plus utiles dans **[Google Colab](https://colab.research.google.com/)**![colab](../../static/img/icons/colab.svg).  

:::tip Astuce
Vous pouvez voir tous les raccourcis disponibles en appuyant sur `Ctrl + M` suivi de `H`.
:::

## Édition et navigation dans une cellule

| Action | Raccourci (Windows/Linux) | Raccourci (Mac) |
| :--- | :--- | :--- |
| **Sauvegarder le notebook** | `Ctrl` + `S` | `⌘` + `S` |
| **Annuler une édition** | `Ctrl` + `Z` | `⌘` + `Z` |
| **Refaire une édition** | `Ctrl` + `Y` | `⌘` + `Shift` + `Z` |
| **Rechercher et remplacer** | `Ctrl` + `H` | `⌘` + `Option` + `F` |
| **Sélectionner un texte** | `Shift` + `⬆️` / `⬇️` / `⬅️` / `➡️` | `Shift` + `⬆️` / `⬇️` / `⬅️` / `➡️` |
| **Sauter des groupes de caractères** | `Ctrl` + `⬅️` / `➡️` | `⌘` + `⬅️` / `➡️`|
| **Sélectionner en sautant des groupes de caractères** |  `Ctrl` + `Shift` + `⬅️` / `➡️` | `⌘` + `Shift` + `⬅️` / `➡️` |
| **Commenter / Décommenter** | `Ctrl` + `/` | `⌘` + `/` |
| **Déplacer la ligne** | `Alt` + `⬆️` / `⬇️` | `Option` + `⬆️` / `⬇️` |
| **Effacer la ligne** | `Ctrl` + `Shift` + `K` | `⌘` + `Shift` + `K` |
| **Multi-curseur dans la cellule** | `Alt` + `Clic` | `Option` + `Clic` |
| **Sélectionner toutes les occurrences dans la cellule** | `Ctrl` + `Shift` + `L` | `⌘` + `Shift` + `L` |

## Exécution et manipulation des cellules

| Action | Raccourci (Windows/Linux) | Raccourci (Mac) |
| :--- | :--- | :--- |
| **Exécuter la cellule** (rester sur la même) | `Ctrl` + `Enter` | `⌘` + `Enter` |
| **Exécuter et passer à la suivante** | `Shift` + `Enter` | `Shift` + `Enter` |
| **Exécuter le code sélectionné** | `Ctrl` + `Shift` + `Enter` | `⌘` + `Shift` + `Enter` |
| **Exécuter les cellules précédentes** | `Ctrl` + `F8` | `⌘` + `F8` |
| **Exécuter toutes les cellules** | `Ctrl` + `F9` | `⌘` + `F9` |
| **Interrompre l'exécution** | `Ctrl` + `M` puis `I` | `⌘` + `M` puis `I` |
| **Annuler la dernière action de cellule** | `Ctrl` + `M` puis `Z` | `⌘` + `M` puis `Z` |
