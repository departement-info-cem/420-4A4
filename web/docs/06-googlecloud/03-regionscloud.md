# 🗺️ Les régions avec GPU L4

Malheureusement, il n'y a pas d'outils pour connaître les régions avec disponibilité de GPU 😞.

Il faut donc faire des essais jusqu'à ce qu'une ressource vous soit attribuée. **Les régions n'ont pas toutes de L4**,
il faut donc esssayer uniquement celles de la liste suivante:

## 🌎 US 
- `us-west (Oregon)`
- `us-central1 (Iowa)`
- `us-east4 (Virginie du Nord)`
- `us-east1 (Caroline du Sud)`
- `us-east4 (Virginie du Nord)`
- `us-west4 (Las Vegas)`

## 🌍 Europe 
- `europe-west1 (Belgique)`
- `europe-west2 (Londres)`
- `europe-west3 (Francfort)`
- `europe-west4 (Pays-Bas)`
- `europe-west6 (Zurich)`

## 🌏 APAC 
- `asia-east1 (Taïwan)`
- `asia-northeast1 (Tokyo)`
- `asia-northeast3 (Séoul)`
- `asia-south1 (Mumbai)`
- `asia-southeast1 (Singapour)`
